# ✨ Features

## 📝 Task Management

* Add new tasks
* Edit existing tasks
* Delete individual tasks
* Clear all tasks instantly

## ✅ Task Completion

* Mark tasks as completed
* Completed tasks show strike-through effect
* Real-time UI updates

## 🔍 Search Functionality

* Search tasks dynamically
* Quickly find tasks from the list

## 📂 Filter System

* View all tasks
* View completed tasks
* View pending tasks

## 💾 LocalStorage Support

* Tasks remain saved even after page refresh
* Persistent browser storage

* Modern glassmorphism design
* Smooth hover animations

# ⚡ JavaScript Functionalities

## ➕ Add Task

* Creates tasks dynamically
* Prevents empty task submission

## ✏️ Edit Task

* Load selected task back into input
* Update tasks easily

## ❌ Delete Task

* Remove tasks instantly
* Updates LocalStorage automatically

## ✅ Complete Task

* Toggle completed state
* Adds line-through effect dynamically

## 💾 Save Tasks

Tasks are stored using:

```javascript
localStorage.setItem("tasks", taskList.innerHTML);
```

This allows tasks to remain saved after page refresh.

## 📂 Filter Tasks

* Show only completed tasks
* Show only pending tasks
* Dynamic task visibility
